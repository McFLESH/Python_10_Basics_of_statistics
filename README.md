Для расчета мер центральной тенденции используют библиотеки pandas, statistics и scipy.stats.

Для просмотра статистической сводки каждого столбца датафрейма в pandas используется метод describe().


Значения можно получить и отдельно для каждого столбца.

Для этого в pandas используются методы mode(), mean(), median(). Например, с помощью data["price"].mean() можно получить среднее арифметическое для столбца "price".


Для расчета мер вариации в Python используют библиотеку numpy.

Размах вычисляется как разница максимального и минимального значений. Для этого используются методы max() и min(). Например, data["price"].max() - data["price"].min().

Для вычисления дисперсии применяется метод var(). Например, np.var(data["price"]).

Стандартное отклонение вычисляется с помощью метода std(). Например, np.std(data["price"]).

Коэффициент вариации определяется отношением стандартного отклонения к среднему значению. Его можно вычислить, например, следующим образом: np.std(data["price"])/np.mean(data["price"]).


В Python можно использовать библиотеку SciPy для расчета p-value. Как уже говорилось ранее, если p-value> 0.05, то распределение признается нормальным.

Для того чтобы получить значение p-value для одной выборки можно использовать функцию ttest_1samp(). В аргументах указывается выборка и ожидаемое среднее значение.
